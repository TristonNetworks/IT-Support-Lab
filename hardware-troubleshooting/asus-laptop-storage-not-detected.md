# Laptop Boot Failure – Storage Device Not Detected

## Device
ASUS Laptop

## Problem
Laptop booted directly into BIOS.
BIOS showed "No Storage Device Present".

## Diagnosis
1. Entered BIOS.
2. Checked storage section.
3. Confirmed SSD was not detected.

## Troubleshooting Steps
1. Powered off laptop.
2. Removed back panel.
3. Located M.2 SSD.
4. Reseated SSD in slot.
5. Restarted system.

## Result
BIOS detected the SSD.
System booted successfully into Windows.
